# 🔮 MagicRPG

Добро пожаловать в **MagicRPG** — хардкорную магическую сборку для Minecraft 1.7.10, где моды не просто существуют рядом, а тесно переплетены друг с другом. 

В этой сборке вам предстоит пройти уникальный путь магического развития. Здесь нельзя просто скрафтить топовую броню в первый день. Вам придется изучать школы магии последовательно, комбинируя знания из Botania, EvilCraft, Blood Magic, Witchery и Thaumcraft, чтобы в конце концов открыть врата в опасные измерения Divine RPG!

## 🌟 Особенности сборки
* **Уникальная прогрессия:** Изменены точки входа в ключевые моды. Изучайте моды шаг за шагом — одна магическая школа открывает двери в следующую.
* **Сбалансированный лейт-гейм:** Измерения Divine RPG заблокированы до тех пор, пока вы не покорите вершины магии (Ихор из Thaumcraft и Дух Гайи из Botania).
* **Стабильность:** Сборка содержит оптимизирующие моды (FastCraft, BetterFps, AI Improvements) и настроена на минимизацию крашей (включено автоудаление багованных блоков в Forge).
* **Никакого мусора:** Убраны дубликаты руд в генерации мира, почищен интерфейс NEI.

## ⚙️ Системные требования и Оптимизация (ОБЯЗАТЕЛЬНО К ПРОЧТЕНИЮ)
Minecraft 1.7.10 очень чувствителен к выделению памяти. **Пожалуйста, не выделяйте игре 10-16 ГБ ОЗУ** — это приведет к зависаниям из-за работы сборщика мусора Java!

* **Рекомендуемая оперативная память (RAM):** строго от `4096 MB` до `6144 MB` (4-6 ГБ).

### Аргументы Java (JVM Arguments)
Для плавной игры и избавления от микро-фризов скопируйте эту строку и вставьте её в настройки вашего лаунчера (раздел "Аргументы Java" / "JVM Arguments"):

`-XX:+UseG1GC -XX:+UnlockExperimentalVMOptions -XX:MaxGCPauseMillis=50 -XX:+DisableExplicitGC -XX:TargetSurvivorRatio=90 -XX:G1NewSizePercent=50 -XX:G1MaxNewSizePercent=80 -XX:G1MixedGCLiveThresholdPercent=35`

* **CurseForge App:** Настройки (шестеренка) -> Minecraft -> Additional Arguments.
* **Официальный лаунчер:** Установки -> MagicRPG -> Больше (More options) -> Аргументы JVM.

## 📥 Установка
minecraft version - 1.7.10
forge version - 10.13.4.1614

### Для одиночной игры (Клиент)
1. Скачайте архив сборки для CurseForge или Modrinth.
2. Импортируйте zip-архив в ваш любимый лаунчер (CurseForge App, Prism Launcher, ATLauncher и др.).
3. Убедитесь, что вы настроили выделение памяти и прописали аргументы Java (см. выше).
4. Запускайте и наслаждайтесь!

### Для создания сервера (Server Pack)
Если вы скачали серверную версию из нашего репозитория:
1. Распакуйте архив в пустую папку.
2. Примите EULA, открыв файл `eula.txt` и изменив `false` на `true`.
3. Для запуска сервера используйте файлы **`start.bat`** (для Windows) или **`start.sh`** (для Linux). В эти скрипты **уже вшиты** правильные аргументы Java и лимиты памяти для идеальной работы сервера!
